# Product Management Materials

This directory contains Product Management (PM) materials used for planning and tracking project development.

## Contents

### Product Requirements Documents (PRDs)
- Feature specifications
- User stories and acceptance criteria
- Technical requirements
- Dependencies and constraints

### OKRs (Objectives and Key Results)
- Quarterly/sprint objectives
- Measurable key results
- Progress tracking

### Roadmaps
- Feature prioritization
- Release planning
- Timeline estimates

### Sprint Planning
- Sprint goals
- Story point estimates
- Team capacity planning

## Purpose

These materials ensure structured product development with clear goals, requirements, and success metrics.

## How to Use

1. Consult PRDs before implementing features
2. Track progress against OKRs
3. Reference roadmap for prioritization decisions
4. Use sprint plans for development cycles
